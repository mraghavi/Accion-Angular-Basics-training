Understanding Dependency Injection (DI) and Services in Angular

In Angular, services are a fundamental way to encapsulate and share logic across your application. Dependency Injection (DI) is the mechanism that allows Angular to inject services into components, making them available for use.

What is a Service?

A service in Angular is typically a class with a specific purpose, such as fetching data, logging, or validation. Services help keep your components clean and focused on their core functionality by handling tasks that are shared across multiple components.

Injecting Services using Dependency Injection (DI)

Angular's Dependency Injection system allows you to inject services into components and other services. This is done by registering the service with Angular's injector and specifying the service in the component's constructor.

Example: Injecting a Basic Service into a Component

  
  // src/app/components/one/one.component.ts
  import {  Component, OnInit }  from '@angular/core';
  import {  BasicService }  from '../../services/basic.service';
  import {  CommonModule }  from '@angular/common';
  @Component({ 
  selector: 'app-one',
  standalone: true,
  imports: [CommonModule],
  templateUrl: './one.component.html',
  styleUrls: ['./one.component.css']
  } )
  export class OneComponent implements OnInit { 
  colors: any;
  
  constructor(private ts: BasicService)  { } 
  
  ngOnInit(): void { 
  this.colors = this.ts.getColors();
  } 
  }

In this example, the `BasicService` is injected into the `OneComponent` through the constructor. The service is then used in the `ngOnInit` lifecycle hook to fetch a list of colors, which are stored in the `colors` property.

Displaying Data in the Template

Once the data is fetched from the service, it can be displayed in the component's template. In this case, we use the `*ngFor` directive to iterate over the list of colors and display each one.

Example: Using `*ngFor` to Display Colors

  
  // src/app/components/one/one.component.html

The `*ngFor` directive iterates over the `colors` array, and each color is displayed within a `div` element. The double curly braces {{ }} are used to interpolate and display the value of each color.

Conclusion

Dependency Injection and services are powerful tools in Angular that promote modularity and code reusability. By encapsulating shared logic in services and injecting them into components, you can create more maintainable and scalable applications.
